### tmux 快捷键

| 切换 panel | 切换 panel 大小 | 切换 window | 切换 session |
| :--------: | :-------------: | :---------: | :----------: |
|    C-j     |      C-x J      |    C-x H    |    C-x (     |
|    C-k     |      C-x K      |    C-x L    |    C-x )     |
|    C-h     |      C-x H      |             |              |
|    C-l     |      C-x L      |             |              |

| 横向分割 panel | 纵向分割 panel |
| :------------: | :------------: |
|     C-x -      |     C-x =      |

| 创建 window | 查看所有 | 查看所有 session | 修改 window 名 | 删除 panel | 放大 panel |
| :---------: | :------: | :--------------: | :------------: | :--------: | :--------: |
|    C-x c    |  C-x w   |      C-x s       |     C-x ,      |   C-x x    |   C-x z    |
